# Metrics Definition

## Cyclomatic Complexity
Measures the number of independent paths through the code.

## Maintainability Index
Composite metric indicating how easy code is to maintain.

## Code Duplication
Percentage of repeated code blocks.

## Test Coverage
Percentage of code executed by tests.

## Defect Density
Number of defects per 1000 lines of code.

## Change Effort
Measured by time, number of files modified, and bugs introduced during a modification task.
